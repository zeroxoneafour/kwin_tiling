Detailed Description

The ClientBuffer class represents a client buffer.

While the ClientBuffer is referenced, it won't be destroyed. Note that the client can still destroy the wl_buffer object while the ClientBuffer is referenced by the compositor. You can use the isDestroyed() function to check whether the wl_buffer object has been destroyed.
