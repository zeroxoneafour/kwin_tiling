◆ createClientList()

void KWin::TabBox::ClientModel::createClientList ( bool partialReset = false)

Generates a new list of Windows based on the current config. Calling this method will reset the model. If partialReset is true the top of the list is kept as a starting point. If not the current active client is used as the starting point to generate the list.

Parameters
partialResetKeep the currently selected client or regenerate everything
