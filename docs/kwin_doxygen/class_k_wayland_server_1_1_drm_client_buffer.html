Detailed Description

The DrmClientBuffer class represents a wl_drm client buffer.

Nowadays, the wl_drm protocol is de-facto deprecated with the introduction of the linux-dmabuf-v1 protocol. Note that Vulkan WSI in Mesa still prefers wl_drm, but that's about to change, https://gitlab.freedesktop.org/mesa/mesa/-/merge_requests/4942/
