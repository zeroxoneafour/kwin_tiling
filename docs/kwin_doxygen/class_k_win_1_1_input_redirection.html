Detailed Description

This class is responsible for redirecting incoming input to the surface which currently has input or send enter/leave events.

In addition input is intercepted before passed to the surfaces to have KWin internal areas getting input first (e.g. screen edges) and filter the input event out if we currently have a full input grab.

Sends an event through all InputFilters. The method function is invoked on each input filter. Processing is stopped if a filter returns true for function.

The UnaryPredicate is defined like the UnaryPredicate of std::any_of. The signature of the function should be equivalent to the following:
