Detailed Description

The SGIVideoSyncVsyncMonitor class monitors vblank events using the GLX_SGI_video_sync extension.

Vblank events are monitored in a separated thread to avoid blocking the main thread. In order to avoid locking up the main X11 connection, the worker thread establishes its own X11 connection.
